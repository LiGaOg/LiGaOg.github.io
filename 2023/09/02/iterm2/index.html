iTerm2下载

官网可以直接下载可执行文件.

终端美化及一些必要效果

字体

字体种类可以选用Nerdfont, 字体中patch了很多icon, 以及表情字体, Nerdfont的官网是: https://www.nerdfonts.com/

  • 推荐的字体是: Noto Nerd Font

配色方案

https://iterm2colorschemes.com/

推荐Dracula

zsh主题

zsh主题可以通过zsh的插件管理器下载, 插件管理器推荐ohmyzsh, 官网是: https://github.com/ohmyzsh/ohmyzsh

主题插件选择powerlevel10k, 官网是: https://github.com/romkatv/powerlevel10k

光标的样式

  • 光标在终端一般模式时需要是|, 并且需要有闪烁效果, 这个|不能太细, 需要粗一些. iTerm2应该不能调节光标的粗细.
  • 光标需要有闪烁效果.
  • 在终端窗口没有被聚焦时, 光标的样式最好时空心的方块.
  • Zsh vi-mode中, 不同的模式下光标的样式需要不同.
  • Cursor guide特效: 在终端下, 光标的这一行的颜色比背景颜色中, 视觉上可以起到分隔的效果.

背景图片

  • 尽量不要在终端中设置透明效果.
  • 选用背景图片, 并且设置图片的渲染度 (Blending).

真颜色

如果在终端中运行如下指令:

awk 'BEGIN{
    s="/\\/\\/\\/\\/\\"; s=s s s s s s s s;
    for (colnum = 0; colnum<77; colnum++) {
        r = 255-(colnum*255/76);
        g = (colnum*510/76);
        b = (colnum*255/76);
        if (g>255) g = 510-g;
        printf "\033[48;2;%d;%d;%dm", r,g,b;
        printf "\033[38;2;%d;%d;%dm", 255-r,255-g,255-b;
        printf "%s\033[0m", substr(s,colnum+1,1);
    }
    printf "\n";
}'

可以输出连续变化的色彩, 那么终端支持真颜色.

如果终端不支持真色彩, 那么有些neovim主题会出问题.

终端功能增强

命令简写

一般常用的有如下几个:

alias s="neofetch"
alias cls="clear"
alias ra="ranger"
alias lg="lazygit"
alias ls="exa --icons"
# remove git plugins in ohmyzsh
alias gc="git clone"
alias ty="open -a typora"
alias config="cd ~/.config"
# enbale true color support for remote
alias ssh="TERM=xterm-256color $(which ssh)"

语法高亮

可执行的命令需要用绿色特殊显示, 不可执行用红色, 可以用zsh的插件zsh-syntax-highlighting:

  • https://github.com/zsh-users/zsh-syntax-highlighting

补全提示

有如下两种补全模式, 对应的zsh插件分别如下

  • 历史补全: https://github.com/zsh-users/zsh-autosuggestions
  • 实时补全: https://mimosa-pudica.net/zsh-incremental.html

vi-mode

vi-mode是用vim的编辑方式编辑当前命令, 需要用bindkey -v开启.

  • vi-mode下, 光标的样式需要有改变, 可以用如下脚本:

    bindkey -v

# b to the beginning of line
bindkey -M vicmd "B" vi-beginning-of-line
# w to the end of line
bindkey -M vicmd "W" vi-end-of-line

# edit cursor style when entering vi-mode
function zle-keymap-select {
    if [[ ${KEYMAP} == vicmd ]] || [[ $1 = 'block' ]]; then
        echo -ne '\e[1 q'
    elif [[ ${KEYMAP} == main ]] || [[ ${KEYMAP} == viins ]] || [[ ${KEYMAP} = '' ]] || [[ $1 = 'beam' ]]; then
        echo -ne '\e[5 q'
    fi
}
zle -N zle-keymap-select

智能选择

  • 最需要的是能够快速选择一个URL, 选择后可以对这个URL复制. iTerm2里可以通过多次双击来快速选中一个URL.
  • 如果终端中出现URL, 需要用浏览器打开时, iTerm2可以用command + 点击URL实现.

复制粘贴

在Mac下一般默认就是command + c/v.

快速跳转

快速跳转到经常cd进入的目录, 这个有一个zsh插件叫z, 在ohmyzsh中内置, 启用后可以在命令行输入z后加Tab, 选择目录进入.

代理设置

可以用all_proxy变量控制命令走代理:

# proxy
po () {
  export all_proxy="socks5h://127.0.0.1:15235"
  echo "Proxy on"
}
pf () {
  unset all_proxy
  echo "Proxy off"
}

在命令行用po, 就可以开启代理. 用pf就可以关闭代理

终端复用

可以使用tmux, 配置文件完全可以采用ohmytmux, 官方网站: https://github.com/gpakosz/.tmux

you-get <视频链接>
